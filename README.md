# Manufacturing Use Case Library

A collection of manufacturing knowledge, business rules, core modules, and practical use cases accumulated from real factory operations over the past 17 years.

## Why this repository?

The most valuable asset in a manufacturing system is not the source code.

It is the knowledge embedded inside business rules, workflows, decision-making processes, and operational experiences.

This repository aims to document and share those learnings.
